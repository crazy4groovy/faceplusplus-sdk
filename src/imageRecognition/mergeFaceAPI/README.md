# API

## CURL

```bash
curl -X POST "https://api-us.faceplusplus.com/imagepp/v1/mergeface" -F "api_key=..." -F "api_secret=..." -F "template_file=@template_file.jpg" -F "template_rectangle=70,80,100,100" -F "merge_file=@merge_file.jpg" -F "merge_rate=70"
```
