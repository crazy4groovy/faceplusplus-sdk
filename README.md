## creds.json

Create this file in the root directory, it's required. Make a [free account](https://console.faceplusplus.com/documents/6329693) to get an [API key and secret](https://console.faceplusplus.com/app/apikey/list).

```json
{
  "api_key": "...",
  "api_secret": "..."
}
```
